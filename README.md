# MyPOS 🏪

Terminal Punto de Venta (TPV) Android con flujo completo 
de pago simulado, construida con Kotlin y Jetpack Compose.

## Funcionalidades
- 🛒 Catálogo de productos con carrito en tiempo real
- 💳 Selección de método de pago — NFC y Tarjeta
- 🔐 Pantalla de autenticación de pago
- ⏳ Simulación de procesamiento de transacción
- ✅ Resultado de pago exitoso o rechazado
- 🔄 Reinicio automático para nueva venta

## Tecnologías
- Kotlin
- Jetpack Compose
- Navigation Compose
- Material Design 3
- Android Studio

## Flujo de la app
```
Productos → Método de Pago → Autenticación → Procesando → Resultado
```

## Cómo correrlo
Clona el repositorio y ábrelo en Android Studio.
